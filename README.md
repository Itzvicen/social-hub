# Social hub 
All your social medias in only page

路 Live Demo: https://weather.vicenweb.es

![Image text](https://github.com/Itzvicen/social-hub/blob/main/preview.png)

##  Built using:

路 ReactJS

路 Tailwind CSS

### Install Dependencies
```sh
$ npm install 
```

### Run development server
```sh 
$ npm run start
```

### Build the project

```sh 
$ npm run build
```
